# Climate Sync Card

Controlează mai multe entități `climate` simultan. Perfect pentru două (sau mai multe) AC-uri identice.

- Trage automat `hvac_modes`, `fan_modes`, `preset_modes`, `swing_modes` și intervalul de temperatură din entități.
- Schimbi mode / preset / fan / swing / temperatură → se aplică pe toate entitățile selectate.
- Editor vizual inclus.

Vezi [README](README.md) pentru configurare completă.
